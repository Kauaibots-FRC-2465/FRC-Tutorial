## What This Lesson Is Really About

Not every variable directly holds its value. Some variables point to an object instead. This lesson teaches that difference because it affects how Java handles text, `null`, and object sharing.

## New Words

- reference: a variable that points to an object
- object: one actual value created from a class
- wrapper type: an object form of a primitive, such as `Integer` or `Double`
- `null`: a special value meaning "this variable is not pointing to any object right now"

## Goals

- Distinguish primitives from references.
- Understand `null` at a practical level.
- Read wrapper types and strings without treating them as mysterious.
