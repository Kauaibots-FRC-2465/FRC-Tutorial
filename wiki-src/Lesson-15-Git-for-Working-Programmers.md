How to use this lesson: use the VS Code Source Control view and Git Graph

## What This Lesson Is Really About

By this point, students are moving between lessons and making their own changes. Git is the tool that keeps that work organized. This lesson focuses on the small set of Git ideas students need every day, using the VS Code Source Control view and Git Graph instead of terminal commands.

## New Words

- branch: a saved line of project history
- Source Control view: the Git view in the VS Code activity bar
- Git Graph: the extension used in this course to view branches and commit history
- stage: mark a change for the next commit
- commit: a saved snapshot of your work

## Goals

- Move safely between lesson branches.
- Read changed files and staged files in the Source Control view.
- Record a small commit with a clear message.
