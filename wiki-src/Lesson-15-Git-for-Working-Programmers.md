# Lesson 15: Git for Working Programmers

Branch: `lesson-15-git-for-working-programmers`
How to use this lesson: use the VS Code terminal for Git commands

## What This Lesson Is Really About

By this point, students are moving between lessons and making their own changes. Git is the tool that keeps that work organized. This lesson focuses on the small set of Git ideas students need every day.

## New Words

- branch: a saved line of project history
- status: Git's report of what changed in your working copy
- diff: a view of what changed in a file
- stage: mark a change for the next commit
- commit: a saved snapshot of your work

## Goals

- Move safely between lesson branches.
- Read `status` and `diff`.
- Record a small commit with a clear message.

- What is tracked, modified, staged, or committed?
