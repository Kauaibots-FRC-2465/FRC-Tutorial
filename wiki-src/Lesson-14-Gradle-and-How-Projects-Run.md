# Lesson 14: Gradle and How Projects Run

## Goals

- Understand what Gradle is doing when the project builds.
- Read `build.gradle` without needing to master every line.
- Use the common project tasks confidently.

## Read First

- `build.gradle`
- `settings.gradle`

## Focus Questions

- Where do dependencies come from?
- Which tasks compile, test, and simulate the project?
- Which errors are build-setup problems and which are Java problems?
