## What This Lesson Is Really About

Programs do not run every line in every situation. They choose paths. This lesson is about following those paths carefully so students can predict what the code will do before they ever run it.

## New Words

- control flow: the order in which the program moves through its code
- branch: one possible path through an `if` statement or `switch`
- loop: code that repeats
- state: the information the program is currently holding

## Goals

- Trace `if`, `switch`, and loop behavior by hand.
- Predict which branch of code runs for a given input.
- Connect control flow to robot decisions.

## Focus Questions

- What path does each input take?
- How many times does each loop run?
- Which cases are named directly, and which fall into the default case?
