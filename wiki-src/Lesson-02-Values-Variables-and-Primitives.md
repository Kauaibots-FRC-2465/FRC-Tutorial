## What This Lesson Is Really About

Robot code constantly works with numbers and true or false decisions. This lesson introduces Java's built-in value types so students can read simple calculations and conditions without treating them as magic.

## New Words

- variable: a named place that stores a value
- primitive type: a built-in Java value type such as `int`, `double`, or `boolean`
- `boolean`: a value that can only be `true` or `false`
- integer division: division that throws away the decimal part

## Goals

- Recognize primitive types.
- See how arithmetic and boolean logic drive robot decisions.
- Understand why integer division can be a bug source.
